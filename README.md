# Ember

A minimalist memory app landing page.

## Features

- **Talk** — Capture a moment through voice
- **Build** — Organize your story
- **Share** — Pass it forward

## Design

- Apple-inspired minimalist aesthetics
- Pitch black background
- SF Pro system typography
- Mobile-first, responsive design
- Safe area support for modern devices

## Usage

Open `index.html` in any modern browser, or serve with:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`
